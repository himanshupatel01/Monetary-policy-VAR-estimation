# Identifying Monetary policy shocks using a Vector Autoregression (VAR) model

## 📌 Project Overview
This project builds a reduced-form and structural **Vector Autoregression (VAR)** system to analyze the dynamic transmission channels of macroeconomic shocks. Using 40 years of quarterly United States economic data (1971 Q1 – 2010 Q4), the model isolates the impact of an unexpected contractionary monetary policy shock, defined as a 1 percentage point increase in the Federal Funds Rate (FFR).

The system tracks 5 key endogenous variables to map out market interactions:
* **Inflation (CPI)**
* **Unemployment Rate**
* **Real GDP Growth Rate**
* **Federal Funds Rate (FFR)**
* **M2 Money Supply Growth**
